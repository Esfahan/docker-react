# React

## Installation
### build

```
$ docker-compose build
```

### Create React App

```
$ create-react-app ${YOUR_APP_NAME}
```

or

```
$ git submodule update -i && git submoudle foreach git pull origin master
```

## Usage
### yarn start

```
$ docker-compose up
```

### Browse
http://localhost:3000

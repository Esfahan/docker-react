# React

## Installation
### build

```
$ docker-compose build
```

### Create React App

```
$ create-react-app ${YOUR_APP_NAME}
```

or Use react-tutorial as a submodule

```
$ git submodule add -f https://github.com/Esfahan/react-tutorial.git code/react-tutorial
$ ./bin/yarn-install ${YOUR_APP_NAME}
```

## Usage
### yarn start

```
$ ./bin/start
```

### Browse
http://localhost:3000
